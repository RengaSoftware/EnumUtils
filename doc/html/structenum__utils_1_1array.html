Detailed Description

template<typename E, typename T>
struct enum_utils::array< E, T >

A basic associative container. It's essentially an O(1) array indexed by enumerations.

The static interface complies to std::array with nothing particularly interesting to it, except for array::make(). The function produces an instance of the class. It's capable of working at compile time as constexpr function.

array.cpp.

static constexpr array make ( )

The core function to generate an array.

By default, it generates an array filled with default-constructed values of type T. The behavior can be customized by providing your own "generator", a functor to produce every item of the array:

#include <enum_utils/array.h>
#include <enum_utils/traits.h>
enum class enumeration { a, b, c, _last = c };
ENUM_UTILS_DEFINE_TRAITS(enumeration, _last)
template <enumeration e>
struct squared_index_generator
{
constexpr auto operator()() const
{
return enum_utils::index(e) * enum_utils::index(e);
}
};
static const auto squared_indices =
enum_utils::array<enumeration, std::size_t>::make<squared_index_generator>(); // { 0, 1, 4 }
static const auto squared_index_of_c = squared_indices[enumeration::c]; // 4
